# SocialReply-Assistant

> 鼠标选中语句，`Ctrl+C` + `Ctrl+Shift+C`，触发情感分析，并给出三句示例回复。

## 项目来源

1. 社交废物人士以己度人。
2. 划词翻译是个方便工具。
